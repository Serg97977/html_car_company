#  S.N. Car Company

Welcome to **S.N. Car Company** — a simple HTML project showcasing different types of cars for sale and rent.

##  About the Project

This is my HTML practice project where I created a car company website.  
The website includes:


Damaged Cars
New Cars
Cars for Rent
Cars from USA
Contact Section
Forms for customer information
Embedded video
Images with lazy loading


The goal of this project was to practice:


HTML structure
Forms and input types
Fieldset & legend
Anchor links
Lists
Semantic layout basics
